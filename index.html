About Us

About Us

Hello, We are two students from Holberton School Tunis and we wanted to challenge ourselves in a project that none of us had done before and the libraries we used were completely new to us.

We Focused more on the p5 JS side of the server before the multiplayer side which was made with Socket IO

This picture is a screenshot of the game in development, we spent lots of hours brainstorming and thinking of the mechanics of the game we need to implement

It was a rough journey with up and downs but at the end of the day we learned alot of new things and we learned how browser multiplayer games function from the inside out!

Features!

Blazing Fast Communication

With Socket IO, the connection between the browser and the server is quick and uninterrupted even with slow connections
